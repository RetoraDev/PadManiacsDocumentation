Platform-agnostic file system access wrapper for reading and writing game data.

[heading: "Description"]

FileSystemTools provides a unified interface for file system operations across different platforms (NW.js, Cordova, Web). It abstracts platform-specific file APIs and provides consistent methods for file management.
