[title: "DialogWindow Class"] ← Back to index

Modal dialog window for user confirmations, prompts, and notifications.

[heading: "Description"]

DialogWindow creates modal dialog boxes with customizable text, buttons, and optional scrolling for long messages. It handles input automatically and provides confirmation/cancel signals.
