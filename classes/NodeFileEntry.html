Node.js implementation of Cordova's FileEntry interface for file operations.

[heading: "Description"]

NodeFileEntry provides a Cordova-compatible FileEntry interface using Node.js file system APIs. It enables consistent file operations across platforms while leveraging native Node.js file handling capabilities.

[heading: "Key Features"] [list] * Cordova FileEntry API compatibility * Node.js file system integration * File reading and writing * MIME type detection * File metadata access [/list] [heading:
