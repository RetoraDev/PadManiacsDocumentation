[title: "Keybindings Class"] ← Back to index

Keybinding configuration interface for keyboard and gamepad controls.

[heading: "Description"]

Keybindings provides a UI for customizing keyboard and gamepad button mappings. Users can remap controls, reset to defaults, and unmap keys by holding ESC.
