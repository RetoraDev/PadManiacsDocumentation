[title: "Player Class"] ← Back to index

Handles all gameplay logic including note processing, scoring, input handling, and skill integration.

[heading: "Description"]

Player is the core gameplay controller that manages note rendering, judgment, scoring, combo tracking, health management, and skill system integration. It uses ChartRenderer for visual rendering.
