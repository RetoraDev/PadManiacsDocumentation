[title: "AddonManager Class"] ← Back to index

Manages the loading, execution, and management of community addons with support for hibernation and safe mode.

[warning: "Be careful modifying this class, you could break the entire game"] [heading: "Description"]

AddonManager is responsible for discovering, loading, and managing all community-created addons. It handles addon assets, behaviors, and provides an interface for enabling/disabling/hibernating addons.

[heading: "Key Features"] [list] * Discover and load addons from storage * Process addon manifests and metadata * Load addon assets and behaviors * Execute addon code in appropriate contexts * Manage addon state (enabled/disabled/hibernating) * Safe mode for error protection * Provide addon management interface * Persistent addon settings [/list] [heading: "Methods"] [table header] [ Method |
