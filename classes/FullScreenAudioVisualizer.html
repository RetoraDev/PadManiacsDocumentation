[title: "FullScreenAudioVisualizer Class"] ← Back to index

Advanced audio visualization system for full-screen display in Jukebox mode.

[heading: "Description"]

FullScreenAudioVisualizer provides real-time audio frequency visualization with multiple visualization styles, smoothing, and customizable colors. It uses the Web Audio API for frequency analysis.

[heading: "Constructor"] [table header] [ Parameter | Type | Description ] [ audioElement | HTMLAudioElement | Audio source for visualization ] [ options | Object | Configuration options ] [/table]
