Displays gameplay results, high score information, and character experience gains after song completion.

[heading: "Description"]

Results shows the player's performance after completing a song, including score, accuracy, combo, and judgments. It handles high score saving, character experience display with animated progress bar, and provides navigation options.

[heading: "Key Responsibilities"] [list] * Display performance statistics * Save high scores if applicable * Show judgment breakdown * Display character experience gain with animation * Show character portrait * Provide navigation options (Next, Continue, Retry, Quit) * Play song preview audio [/list] [heading: "Constructor"] [table header] [ Parameter | Type | Description ] [ gameData | Object |
